## Опис проекту
Ця програма демонструє використання методів класу `CompletableFuture` для асинхронної роботи в Java.Основні методи:

- `supplyAsync()`
- `thenApplyAsync()`
- `thenAcceptAsync()`
- `thenRunAsync()`

## Основний функціонал
1. **Генерація матриці**:
   - Матриця розміром 3x3 генерується випадковими числами від 1 до 100 асинхронно.
   - Час виконання задачі виводиться на екран.

2. **Вивід матриці**:
   - Згенерована матриця виводиться у зручному форматі асинхронно.
   - Час виконання задачі виводиться на екран.

3. **Вивід стовпчиків матриці**:
   - Кожен стовпчик матриці виводиться окремо з повідомленням у форматі:
     ```
     Стовпчик N: значення1, значення2, значення3
     ```
   - Час виконання задачі виводиться на екран.

4. **Фінальна задача**:
   - Метод `thenRunAsync()` демонструє виконання завершальної дії після завершення всіх попередніх задач.
   - Виводиться повідомлення про успішне завершення.


## Очікуваний результат
При запуску програма має:
1. Згенерувати матрицю 3x3 і вивести її.
2. Асинхронно вивести кожен стовпчик матриці.
3. Після цього виконати фінальну задачу з повідомленням про завершення всіх задач.
4. Показати час виконання кожної з задач.
